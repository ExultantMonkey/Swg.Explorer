A simple archive (TRE file) viewer/explorer for Star Wars Galaxies.

Requires the Microsoft XNA Framework Redistributable 4.0.

Features:

  • View multiple TRE archives in a combined view
  • View and preview media and data files; including EA Interchange File Format (IFF), and String Table Format (STF)
  • Extract data files
  • Export static meshes to Collada (dae) model files
  • Export string files to CSV text files

Uses:

  • DDSLib by Popescu Alexandru Cristian
