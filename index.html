Swg.Explorer

Download this project as a .zip file Download this project as a tar.gz file

Swg.Explorer (v0.2) is a a simple archive (TRE file) viewer/explorer for Star Wars Galaxies.

Features

  • View multiple TRE archives in a single combined view
  • View and preview media and data files; including EA Interchange File Format (IFF), and String Table Format (STF)
  • Extract data files
  • Export meshes (static and dynamic) to Collada (dae) model files
  • Export string files to CSV text files
